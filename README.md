# 🎯 AdVision-AI

An end-to-end Machine Learning web application that predicts whether a customer is likely to click an online advertisement using Logistic Regression.

## 🌐 Live Demo

🚀 https://advision-ai-191u.onrender.com

## 💻 GitHub Repository

https://github.com/vanshi13rathore/AdVision-AI

---

# 📌 Project Overview

AdVision-AI predicts the probability of a customer clicking an online advertisement based on customer behavior and demographic information.

The application provides:

- 🎯 Advertisement Click Prediction
- 📊 Prediction Probability
- 💼 Business Recommendation
- 📄 Downloadable PDF Report

# 📊 Dataset

Dataset contains customer advertisement behavior including:

- Daily Time Spent on Site
- Age
- Area Income
- Daily Internet Usage
- Gender
- Clicked on Ad

Dataset Size:

- 1000 Records

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/vanshi13rathore/AdVision-AI.git
```

Move into project

```bash
cd AdVision-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app/streamlit_app.py
```
