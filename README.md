# 🧠 AdVision-AI

> **AI-Powered Advertisement Click Prediction using Machine Learning**

AdVision-AI is an end-to-end Machine Learning web application that predicts whether a customer is likely to click on an online advertisement. It combines data preprocessing, Logistic Regression, interactive dashboards, and a modern Streamlit interface to help businesses make data-driven marketing decisions.

## 📌 Project Overview

Online advertising is one of the most effective marketing strategies, but showing advertisements to every customer is inefficient and costly.

AdVision-AI predicts whether a customer is likely to click on an advertisement based on customer behavior and demographic information. Businesses can use these predictions to improve campaign targeting and maximize return on investment (ROI).

---

## 🎯 Problem Statement

Predict whether a customer will click on an online advertisement using Machine Learning.

The prediction is based on the following customer information:

- Daily Time Spent on Site
- Age
- Area Income
- Daily Internet Usage
- Gender

# ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/vanshi13rathore/AdVision-AI.git
```

### Move into the project

```bash
cd AdVision-AI
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the environment

#### Windows

```bash
venv\Scripts\activate
```

#### macOS/Linux

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app/streamlit_app.py
```
